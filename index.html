APUNTES NODE JS

Comandos Node.js

confirmarmar que esta instalado

node --version

Cargar NPM (Node Package Manager) en el proyecto

nom init
npm init -y

Consultar lo que regresa nuestro archivo js por consola 

node nombre_archivo.js

Exportar funciones de un archivo js


const  sumar        =       (a,b)   => a+b;
const  restar       =       (a,b)   => a-b;
const  dividir      =       (a,b)   => a/b;
const  multiplicar  =       (a,b)   => a*b;
                                                    
                                                    
                                                    
exports.sumar = sumar; 
exports.restar = restar; 
exports.dividir = dividir; 
exports.multiplicar = multiplicar;

Exportar un modulo

Utilizaremos la palabra reservada module.export = y el nombre de nuestro objeto o función.



const math = {};

const  sumar        =       (a,b)   => a+b;
const  restar       =       (a,b)   => a-b;
const  dividir      =       (a,b)   => a/b;
const  multiplicar  =       (a,b)   => a*b;
                            
                            
                            
math.sumar = sumar; 
math.restar = restar; 
math.dividir = dividir; 
math.multiplicar = multiplicar; 
                            
module.export = math;
